# BedMed

**BedMed** est une application de supervision d'une chambre de patient via une tablette connectée.

## Description

Ce projet vise à améliorer la surveillance des patients hospitalisés en fournissant une interface intuitive pour les soignants. Grâce à une tablette connectée, le personnel médical peut surveiller divers paramètres et interagir avec l'environnement du patient.

## Fonctionnalités

- Surveillance en temps réel des signes vitaux du patient
- Contrôle des dispositifs médicaux connectés
- Alertes et notifications en cas d'anomalies
- Interface utilisateur adaptée aux besoins du personnel médical

## Technologies utilisées

- **PHP**
- **CSS**
- **HTML**
- **JavaScript**
